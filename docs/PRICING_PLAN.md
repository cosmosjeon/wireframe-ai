# Vibeframe AI 와이어프레임 생성기 - 종량제 요금제 기획서

> **작성일**: 2025-01-20  
> **버전**: 2.0 (서비스 특성 반영)  
> **서비스 정의**: AI 기반 와이어프레임/UI 설계 도구 (Excalidraw 기반)

---

## 1. Executive Summary

Vibeframe은 **채팅 기반 AI 와이어프레임 생성 도구**입니다. 사용자가 원하는 UI를 설명하면 AI가 Excalidraw 형식의 와이어프레임을 생성하고, 반복적인 대화를 통해 디자인을 개선할 수 있습니다.

### 핵심 가치 제안
- **속도**: 몇 분 만에 와이어프레임 초안 완성
- **접근성**: 디자인 경험 없이도 UI 설계 가능
- **반복성**: 대화로 쉽게 수정/개선
- **호환성**: Excalidraw JSON 내보내기로 타 도구 연동

### 과금 전략
**"생성 횟수(Generations)" 기반 종량제 + 구독 하이브리드**

## 4. Vibeframe 요금제 설계

### 4.1 과금 단위: Generation (생성)

#### Generation이란?
- **1 Generation = 1회 AI 와이어프레임 생성/수정 요청**
- 사용자가 채팅에 요청을 보내고 AI가 와이어프레임을 생성/수정하면 1 Generation 소비
- 단순 질문(설명 요청 등)은 0.5 Generation

#### Generation 소비 예시

| 사용자 요청 | 소비량 | 설명 |
|------------|--------|------|
| "로그인 페이지 만들어줘" | 1 Gen | 새 와이어프레임 생성 |
| "버튼을 오른쪽으로 옮겨줘" | 1 Gen | 기존 요소 수정 |
| "네비게이션 바 추가해줘" | 1 Gen | 요소 추가 |
| "이 레이아웃 설명해줘" | 0.5 Gen | 설명만 요청 |
| "대시보드 전체 디자인해줘" | 1-2 Gen | 복잡한 생성 |

## 6. 사용자 시나리오

### 6.1 Free 사용자 여정

```
1일차: 가입 → 3개 와이어프레임 생성 (3 Gen)
2일차: 수정 작업 (5 Gen - 일일 한도)
3일차: 수정 작업 (5 Gen)
4일차: 새 프로젝트 시도 (5 Gen)
...
2주차: 월 한도 20회 소진
     → "Pro로 업그레이드하면 200회 사용 가능" 안내
     → 전환 또는 다음 달 대기
```

### 6.2 Pro 사용자 여정

```
주간 사용량: 약 40-50 Generations
- 새 프로젝트 시작: 10 Gen
- 반복 수정: 30 Gen
- 탐색/실험: 10 Gen

월말: 180/200 Gen 사용
    → 미사용 20회 중 50회까지 이월 가능
    → 다음 달 220회로 시작
```

### 6.3 Team 전환 트리거

```
Pro 사용자가:
- 팀원과 공유 필요 발생
- 월 200회 초과 사용
- 브랜드 일관성 유지 필요

→ Team 플랜 전환 ($35/인)
→ 500 Gen/인 + 팀 공유 + 브랜드 킷
```

## 11. 결론

Vibeframe의 **Generation 기반 종량제 + 구독 하이브리드** 모델은:

1. **공정성**: 와이어프레임 생성/수정 횟수로 명확한 가치 측정
2. **경쟁력**: Uizard/Visily와 동급 가격대, 더 관대한 한도
3. **확장성**: 개인(Pro) → 팀(Team) → 기업(Enterprise) 자연스러운 성장 경로
4. **예측 가능성**: 월 포함량 + 필요시 추가 구매

**핵심 차별화**: 
- Excalidraw 기반 → 오픈 포맷, 타 도구 연동 용이
- 채팅 기반 반복 수정 → 더 직관적인 UX
- 다중 LLM 지원 → 사용자 선택권
